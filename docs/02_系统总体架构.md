## 2.2 技术选型

| 层级 | 技术组件 | 选型理由 |
|------|---------|---------|
| **前端框架** | React 19 + UmiJS Max 4 + Ant Design 5 Pro | 企业级中后台最佳实践，Pro Components 开箱即用 |
| **图可视化** | AntV G6 v4 + react-force-graph-2d | 高性能图渲染，支持分层布局、节点下钻、路径高亮 |
| **后端 API** | FastAPI (Python 3.13) | 异步高性能，与 ML/NLP 生态无缝集成 |
| **图数据库** | Neo4j 5.x (Bolt 协议) | 原生图存储，Cypher 查询语言，支持百亿级规模 |
| **大模型网关** | DeepSeek API / OpenAI 兼容协议 | 模型可插拔，支持私有化部署 |
| **NLP 管线** | spaCy 3.8 + LLM-based NER | 实体抽取双引擎（规则+大模型），灵活切换 |
| **多智能体编排** | 自研 DRA Engine (LangGraph 兼容) | 支持顺序/辩论/反思多模式，人在回路 |
| **数据采集** | Scrapy + httpx + 自定义爬虫框架 | 成熟生态，支持分布式采集 |
| **向量检索** | NumPy + scikit-learn (TfIdf/余弦相似度) | 轻量级实体对齐，可升级至向量数据库 |
