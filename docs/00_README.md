## 核心概念速览

```
数据采集层 → 知识计算层 (KG) → 认知推理层 (Multi-Agent) → 业务应用层
     ↑                                                          │
     └──────────────── 反馈闭环 (Human-in-the-loop) ────────────┘
```

- **四层图谱：** 主体层 → 事件层 → 特征层 → 法规/行动层
- **多智能体：** 检索 Agent → 分析 Agent → 合规 Agent → 汇报 Agent
- **人在回路：** 业务专家可在任意环节介入、修正、审核
