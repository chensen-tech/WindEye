一、任务说明
============================================================

本知识库用于帮助 Dify 工作流从资本市场监管文件、上市公司公告、处罚决定、问询函、IPO 发行材料、舆情文本等材料中识别风险类型。

需要识别的五类风险类型为：

1. 高价超募
   风险编码：OVERPRICED_OVERFUNDING

2. 抱团压价
   风险编码：COLLUSIVE_UNDERPRICING

3. 信息披露违规
   风险编码：DISCLOSURE_VIOLATION

4. 操纵市场
   风险编码：MARKET_MANIPULATION

5. 重大舆情
   风险编码：MAJOR_PUBLIC_OPINION

识别要求：
- 一个文件可以命中一个或多个风险类型。
- 必须给出命中的关键词、判断依据、原文证据和置信度。
- 如果风险类型不明确，应返回“未识别”或标记 need_human_review=true。
- 不得编造文本中不存在的信息。
- 风险类型识别结果应作为后续事件提取、风险特征提取和知识图谱构建的输入。

三、风险类型一：高价超募
============================================================

1. 风险定义

高价超募是指上市公司实际募集资金超过招股文件中计划募集资金额，同时发行价格、发行市盈率、估值指标或募集资金规模处于较高水平，且估值合理性和审慎性不足的风险情形。

高价超募通常发生在 IPO 或注册制发行场景中。其核心判断包括两个方面：一是实际募集资金是否超过计划募集资金；二是发行估值、发行价格、市盈率等是否明显偏高，以及估值结论是否经过充分、合理、审慎论证。

2. 典型识别关键词

高价超募、超募、实际募集资金、计划募集资金、预计募集资金、招股文件、招股意向书、募集资金净额、发行价格、发行价、市盈率、发行市盈率、估值指标、估值结论、投资价值研究报告、估值合理性、估值审慎性、行业平均水平、可比公司平均水平、发行价格较高、估值偏高、募集资金超过预期、超额募集、注册制、新股发行、IPO 发行、二级市场资金抽离

3. 关键判断依据

判断高价超募时，应重点关注以下证据：

（1）实际募集资金是否超过招股文件或招股意向书中预计募集资金。
（2）发行价格、市盈率、每股价格等是否处于市场较高水平。
（3）估值指标是否超过行业或可比公司平均水平。
（4）投资价值研究报告是否充分论证估值结论的合理性和审慎性。
（5）超募资金是否存在非必要投资、低效率使用或资金闲置风险。
（6）发行估值是否导致后市下行压力或对市场资金形成非正常抽离。
（7）文本中是否同时出现“实际募集资金超过计划募集资金”和“估值偏高/发行价偏高/市盈率偏高”等证据。

4. 常见触发表述

- “实际募集资金超过招股意向书预计募集资金。”
- “发行市盈率明显高于行业平均水平。”
- “估值结论对应估值指标超过可比公司平均水平。”
- “发行价格处于市场较高水平。”
- “募集资金净额远高于原计划募集金额。”
- “应当充分评估论证估值结论的合理性和审慎性。”
- “超募资金使用效率较低。”
- “发行定价偏高，可能增加后市下行压力。”

法规层 Regulation：证券发行与承销规则、投资价值研究报告关注事项、交易所发行承销自律规则、证监会发行监管规则

7. 分类规则

如果文本同时满足以下条件，应优先判断为“高价超募”：
- 出现实际募集资金、预计募集资金、计划募集资金、超募等信息；
- 出现发行价格、市盈率、估值指标、估值合理性、估值审慎性等信息；
- 文本说明实际募集资金超过计划募集资金，或估值明显高于行业/可比公司水平。

如果只出现“募集资金”但没有“超募”或“估值偏高”证据，置信度应降低，并标记 need_human_review=true。

四、风险类型二：抱团压价
============================================================

1. 风险定义

抱团压价是指在新股询价过程中，多个询价机构或投资者通过协商、默契或一致报价等方式，故意压低新股发行价格，以获得更高认购利益或价格优势的行为。

该风险通常发生在 IPO 网下询价、发行定价和机构报价过程中。其核心特征是多个机构报价高度一致，报价明显偏低，或者存在内部沟通、协调报价、默契压价等证据。

2. 典型识别关键词

抱团压价、压价、询价机构、网下投资者、机构投资者、报价一致、报价高度一致、报价集中、低价报价、压低发行价、发行价格偏低、IPO 询价、新股询价、协商报价、默契报价、机构报价、低价发行、市值接近上市标准、最低发行市盈率、最低募资金额、报价区间集中、内部沟通、协调报价、联合报价、报价与市场价值偏离

3. 关键判断依据

判断抱团压价时，应重点关注：

（1）多个询价机构报价是否高度一致。
（2）报价是否集中在某一特定价格或价格区间。
（3）报价是否明显低于新股实际市场价值或行业平均估值。
（4）是否存在机构投资者之间内部沟通、协调或默契报价证据。
（5）发行价是否接近最低上市标准，或呈现异常偏低特征。
（6）报价行为是否可能违反市场公平竞争原则。
（7）是否出现“多家机构一致报价”“几乎所有报价集中于某一价格”等表述。

4. 常见触发表述

- “多家询价机构一致报价。”
- “大部分机构报价集中在同一价格区间。”
- “发行价明显低于市场合理价值。”
- “报价低于行业平均估值水平。”
- “机构投资者之间存在协商或协调报价。”
- “共同提出较低报价。”
- “发行价刚好超过上市最低市值标准。”
- “多个机构通过默契压低发行价格。”

confidence：置信度

6. 图谱构建映射

主体层 Subject：发行人、询价机构、机构投资者、网下投资者、保荐机构、承销商
事件层 Event：IPO 询价事件、低价发行事件、一致报价事件、机构协同报价事件、抱团压价事件
特征层 Feature：报价高度一致、报价集中、报价明显偏低、低于市场合理价值、机构协同嫌疑、发行价接近最低上市标准、市场公平竞争受损
法规层 Regulation：证券发行与承销规则、网下投资者管理规则、交易所自律监管规则、反不正当竞争或市场公平相关规范

7. 分类规则

如果文本中同时出现以下证据，应判断为“抱团压价”：
- 多个询价机构、网下投资者、机构投资者；
- 报价一致、报价集中、共同低价报价；
- 发行价格明显偏低或接近最低上市标准；
- 存在内部沟通、协商、协调或默契报价证据。

如果只是发行价较低，但没有多机构一致报价或协同证据，应降低置信度。

五、风险类型三：信息披露违规
============================================================

1. 风险定义

信息披露违规是指上市公司未按照法律法规要求真实、准确、完整、及时、公平地披露重要信息，或者在信息披露文件中存在虚假记载、误导性陈述、重大遗漏、不及时披露、不正当披露等行为。

信息披露违规通常涉及公告、定期报告、临时报告、财务报告、监管处罚决定、交易所问询函等材料。其核心特征是不真实、不准确、不完整、不及时、不规范。

2. 典型识别关键词

信息披露违规、信息披露违法、虚假记载、误导性陈述、重大遗漏、不及时披露、延迟披露、提前披露、不正当披露、不完整披露、不准确披露、不真实披露、未按规定披露、隐瞒重大事项、财务造假、虚增收入、虚增利润、虚列资产、虚报成本费用、虚减资产、虚增资产、提前确认收入、虚构客户、虚构合同、虚假交易、关联交易、资金占用、会计政策使用不当、坏账计提、股权投资核算、内部购销、受托经营、信息披露管理办法、证券法、定期报告、临时公告、年报、半年报、问询函、监管函、处罚决定

3. 主要违规类型

（1）误导性陈述：信息披露文件中的某些记载虽可能具有部分真实性，但由于表述方式存在缺陷，容易导致投资者误解。
（2）虚假记载：信息披露文件记载与事实不符，包括客观上没有发生的事项被写入文件，或真实事项被歪曲。
（3）重大遗漏：上市公司遗漏对投资者决策具有重要影响的信息，包括故意遗漏和过失遗漏。
（4）不正当披露：即使披露内容真实，也可能因为披露方式、披露时间、披露地点不合规而构成违规。
（5）不及时披露：应当披露的信息未在规定时间内披露，包括延迟披露和未及时补充披露。
（6）财务造假：通过虚增收入、虚增利润、虚列资产、制造虚假交易、会计政策滥用等方式影响财务信息真实性。

4. 关键判断依据

判断信息披露违规时，应重点关注：

（1）披露内容是否真实。
（2）披露内容是否准确。
（3）披露内容是否完整。
（4）披露是否及时。
（5）披露方式是否符合规定。
（6）是否存在虚假记载、误导性陈述或重大遗漏。
（7）是否存在虚增收入、虚增利润、虚列资产、虚假交易等财务造假手段。
（8）是否违反《证券法》《上市公司信息披露管理办法》或交易所信息披露规则。
（9）是否被监管机构出具监管函、问询函、纪律处分、行政处罚决定。

5. 常见触发表述

- “未按规定及时披露。”
- “存在虚假记载。”
- “存在误导性陈述。”
- “存在重大遗漏。”
- “信息披露不真实、不准确、不完整。”
- “虚增收入和利润。”
- “虚列资产。”
- “提前确认收入。”
- “虚构客户或销售合同。”
- “会计政策使用不恰当。”
- “未披露重大关联交易。”
- “隐瞒资金占用事项。”
- “违反上市公司信息披露管理办法。”

事件层 Event：信息披露违规事件、财务造假事件、虚假记载事件、重大遗漏事件、不及时披露事件、监管问询事件、行政处罚事件
特征层 Feature：披露不真实、披露不准确、披露不完整、披露不及时、虚增收入、虚增利润、虚列资产、虚构交易、重大事项遗漏、投资者误导
法规层 Regulation：《中华人民共和国证券法》《上市公司信息披露管理办法》上交所信息披露工作评价指引、深交所信息披露工作评价指引、交易所自律监管规则

8. 分类规则

如果文本中出现上市公司公告、披露文件、监管处罚，并包含“虚假记载、误导性陈述、重大遗漏、不及时披露、未按规定披露、财务造假”等内容，应判断为“信息披露违规”。

如果文本主要是负面舆情但没有披露义务、披露文件或监管依据，应避免误判为信息披露违规，可转向“重大舆情”。

六、风险类型四：操纵市场
============================================================

1. 风险定义

操纵市场是指行为人利用资金优势、持股优势、信息优势、账户控制优势或其他手段，通过连续交易、约定交易、对倒交易、幌骗交易、蛊惑交易、抢帽子交易、跨市场操纵等方式，影响或意图影响证券交易价格或交易量，从而谋取利益、规避损失或制造市场假象的行为。

操纵市场的认定通常围绕“行为手段 + 影响”或“行为手段 + 意图影响”展开。

2. 典型识别关键词

操纵市场、操纵行情、市场操纵、连续交易、约定交易、对倒交易、洗售交易、幌骗交易、虚假申报、频繁申报、频繁撤单、大量申报并撤销、蛊惑交易、抢帽子交易、反向交易、跨市场操纵、集中资金优势、持股优势、信息优势、账户组、实际控制账户、联合买卖、连续买卖、价格操纵、交易量操纵、影响证券交易价格、影响证券交易量、意图影响、诱导投资者、虚假重大信息、市值管理、价量异常、高频交易、扰乱交易、证券法第五十五条

3. 操纵市场主要类型

（1）连续交易操纵：通过连续买卖特定证券，影响证券交易价格或交易量。
（2）约定交易操纵：与他人串通，以事先约定的时间、价格和方式进行证券交易。
（3）对倒交易操纵：在自己实际控制的账户之间进行证券交易，制造交易假象。
（4）幌骗交易操纵：不以成交为目的，频繁或者大量申报并撤销申报，扰乱市场交易秩序。
（5）蛊惑交易操纵：利用虚假或者不确定的重大信息诱导投资者进行交易。
（6）抢帽子交易操纵：对证券、发行人公开作出评价、预测或投资建议，并进行反向证券交易。
（7）跨市场操纵：利用其他相关市场的活动操纵证券市场。
（8）其他方式操纵：通过其他影响或意图影响证券交易价格或交易量的行为进行操纵。

4. 关键判断依据

判断操纵市场时，应重点关注：

（1）是否存在集中资金优势、持股优势、信息优势。
（2）是否存在账户组或实际控制多个账户。
（3）是否存在连续买卖、约定交易、对倒交易、洗售交易。
（4）是否存在频繁申报并撤销申报，且不以成交为目的。
（5）是否传播虚假或不确定重大信息诱导投资者交易。
（6）是否公开荐股、预测或评价后进行反向交易。
（7）是否影响或意图影响证券交易价格或交易量。
（8）是否通过价差、减持、市值管理、协议安排等方式获利。
（9）是否有交易前后异常行为作为主观意图证据。

============================================================
七、风险类型五：重大舆情
============================================================

1. 风险定义

重大舆情是指在网络空间或媒体传播环境中，公众、权威媒体、监管机构、意见领袖、投资者等围绕上市公司、金融机构、行业风险、监管事项或突发事件形成广泛关注、讨论和传播，并可能影响相关主体声誉、市场预期、投资者行为或市场稳定的风险情形。

重大舆情通常具有传播速度快、影响范围广、难以控制、互动程度高等特征。

2. 典型识别关键词

重大舆情、舆情风险、网络舆情、舆论热点、负面舆情、负面报道、媒体报道、权威媒体、官方回应、意见领袖、公众讨论、投资者关注、社交媒体、话题发酵、传播速度快、影响范围广、难以控制、互动程度高、声誉风险、市场情绪、股价波动、澄清公告、回应舆情、舆论关注、热点事件、突发事件、负面信息、转载传播、集中讨论

3. 关键判断依据

判断重大舆情时，应重点关注：

（1）是否在短时间内迅速传播。
（2）是否引发公众、媒体、投资者或意见领袖大量讨论。
（3）是否被权威媒体、官方机构或监管机构关注。
（4）影响范围是否较广，是否波及相关主体、行业或市场。
（5）是否影响公司声誉、投资者预期或股价表现。
（6）是否存在持续发酵、难以控制的传播趋势。
（7）是否引发公司澄清、监管关注或信息披露义务。

4. 常见触发表述

- “事件引发市场广泛关注。”
- “相关话题在网络平台迅速传播。”
- “多家媒体进行了报道。”
- “投资者集中讨论。”
- “舆情持续发酵。”
- “公司声誉受到影响。”
- “相关负面报道引发股价波动。”
- “公司发布澄清公告。”
- “官方媒体和意见领袖转发关注。”
- “公众参与互动程度高。”

7. 分类规则

如果文本中出现“媒体报道、网络传播、公众讨论、舆情发酵、声誉风险、投资者关注、股价影响”等证据，应判断为“重大舆情”。

如果文本已经明确涉及虚假记载、重大遗漏或未及时披露，则可能同时命中“信息披露违规”和“重大舆情”。

============================================================
八、多标签识别规则
============================================================

一个文件可能同时涉及多个风险类型。识别时应允许多标签输出。

常见组合：

1. 信息披露违规 + 重大舆情
如果上市公司未及时披露重大事项，并引发媒体报道、投资者关注或网络发酵，应同时识别为“信息披露违规”和“重大舆情”。

2. 高价超募 + 重大舆情
如果 IPO 高价发行或超募引发市场争议、媒体关注和投资者讨论，应同时识别为“高价超募”和“重大舆情”。

3. 抱团压价 + 重大舆情
如果机构报价高度一致或发行价异常偏低，并引发媒体报道或市场质疑，应同时识别为“抱团压价”和“重大舆情”。

4. 操纵市场 + 信息披露违规
如果行为人利用虚假信息诱导交易，同时上市公司或相关主体存在披露不真实、不完整、不及时行为，应同时识别为“操纵市场”和“信息披露违规”。

5. 操纵市场 + 重大舆情
如果异常交易、账户组、市场操纵行为引发广泛舆论关注或媒体报道，应同时识别为“操纵市场”和“重大舆情”。

============================================================
九、置信度判定规则
============================================================

高置信度：0.85-1.00
条件：
- 文本中存在明确风险名称或明确行为描述；
- 同时存在主体、行为、证据和判断依据；
- 原文中有监管处罚、问询、公告或量化指标支撑。

中等置信度：0.60-0.84
条件：
- 文本出现多个相关关键词；
- 有行为描述，但证据链不完整；
- 需要进一步补充主体、指标或法规依据。

低置信度：0.30-0.59
条件：
- 只出现零散关键词；
- 缺少明确主体或行为；
- 可能属于其他风险类型；
- 应标记 need_human_review=true。

未识别：0.00-0.29
条件：
- 文本没有明显资本市场风险信息；
- 没有五类风险相关证据；
- 不应强行分类。

十五、注意事项
============================================================

1. 不要把“重大舆情”误判为“信息披露违规”。
只有当文本明确涉及披露义务、披露文件、虚假记载、重大遗漏、不及时披露等内容时，才识别为信息披露违规。

2. 不要把“股价波动”直接判断为“操纵市场”。
操纵市场需要行为手段、账户组、交易行为、影响价格或交易量等证据。

3. 不要把“发行价高”直接判断为“高价超募”。
高价超募需要同时关注实际募集资金超过计划募集资金、发行估值偏高或估值合理性不足。

4. 不要把“发行价低”直接判断为“抱团压价”。
抱团压价需要多个询价机构报价一致、报价集中或协同压价证据。

5. 一个文件可以多标签识别。
例如信息披露违规引发广泛媒体报道，可以同时识别为信息披露违规和重大舆情。

6. 低置信结果必须标记 need_human_review=true。
